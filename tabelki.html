Elementy tabelki

Do tworzenia tabelki słyżu element "table". Zawartość jest zapisywana jako kolejne wiersze.
Element "tr" tworzy nowy wiersz tabeli. (z ang. table row - rząd tabeli) Koniec rzędu to "/tr"
Każda komórka tabeli jest reprezentowana przez element "td" (z ang. table data) Koniec komórki to "/td"

Nagłówki tabel

Element "th" służy do oznaczania nagłowków rzędów i kolumn. (z ang. table heading)
Jeśli chcemy pustą komórkę to zostawiamy "td" lub "th" puste.

Łączenie komórek kolumn

Aby połączyć komórki w sąsiadujących kolumnach (lewa + prawa / poziome) należy użyc atrybutu ' colspan="x" ' x - liczba określająca ile kolumn na zajmować dana komórka.

Łączenie komórek rzędów

Aby połączyć komórki w sąsiadujących rzędach (góra + dół/ pionowe) należy użyc atrybutu ' rowspan="x" ' x - liczba określająca ile rzędów na zajmować dana komórka.
